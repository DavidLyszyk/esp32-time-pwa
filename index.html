Arduino IDE Setup
🛠 Steps to Flash
  1. Install esp32 board package via Board Manager (URL: espressif/arduino-esp32)
  2. Select board: ESP32C3 Dev Module
  3. Set USB CDC On Boot: Enabled (for Serial.println)
  4. No extra BLE library needed — it's bundled with the esp32 package
  5. Flash, open Serial Monitor at 115200 baud
